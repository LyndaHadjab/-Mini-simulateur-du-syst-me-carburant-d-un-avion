# Mini-simulateur-du-systeme-carburant-d-un-avion
le système de carburant gère plusieurs opérations, vidange, ainsi que les flux de carburant entre le moteur et les réservoirs pendant le vol. Il permet aussi de transférer le carburant d’un réservoir à l’autre. Le but du projet est d’implémenter un simulateur pour ce système afin que les pilotes puissent s’entraîner avant d’aller sur le terrain.
